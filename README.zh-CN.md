#### 简介
一个remote bitcoin 钱包。使用 blockstream API 进行广播交易和获取链上信息。目前只在**测试网**进行了发送和接收BTC测试。如果你在主网使用该钱包，后果自负。

#### 功能
- [x] 生成钱包账号
- [x] 恢复钱包
- [x] 发送和接收BTC
- [x] 展示交易活动
- [x] 地址簿

#### 如何构建?
- 安装 `Rust` 和 `Cargo`
- 运行 `make`
- 参考 [Makefile](./Makefile) 了解更多信息
